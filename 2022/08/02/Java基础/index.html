变量

概述: 在程序运行过程中,其值是可以发生改变的量

作用: 变量其实就是用来存储数据的,并且里面存储的数据是可以发生改变的

格式: 数据类型 变量名 = 值;

数据类型: byte,short,int,long,float,double,char,boolean,String

变量名: 自己定义即可

值: 要存储的数据

类型转换

自动类型转换:

概述: 把一个表示数据范围小的数值或者变量赋值给一个表示数据范围大的变量

格式: 表示数据范围大的数据类型 变量名 = 表示数据范围小的值;

强制类型转换:

概述: 把一个表示数据范围大的数值或者变量赋值给一个表示数据范围小的变量

格式:表示数据范围小的数据类型 变量名 = (表示数据范围小的数据类型)表示数据范围大的值;

注意事项（1）

数据范围大小关系: byte < short < int < long < float < double

数据范围大小关系: char < int < long < float < double

强制类型转换会发送数据丢失,故慎用

boolean类型不能发生类型转换

char,byte,short发生数学运算,会自动提升为int类型

算术运算符与选择结构

算术运算符

  • 表达式: 使用运算符把常量或者变量连接起来的符号java语法的式子,就是表达式
  • 算术运算符:
  • -
  • / 得到的是商,2个整数相除得到的一定是整数,如果想得到小数,必须有浮点数参与
  • % 得到的是余数
  • 案例

字符串的+操作

  • 任何数据和字符串相加都会拼接成一个新的字符串(运算方式从左往右)

赋值运算符

  • = 赋值
  • += 先加再赋值,隐含了强制类型转换
  • -= 先减再赋值,隐含了强制类型转换
  • *= 先乘再赋值,隐含了强制类型转换
  • /= 先除再赋值,隐含了强制类型转换
  • %= 先取余再赋值,隐含了强制类型转换

自增自减运算符

  • 自增: ++
  • 单独使用: 无论是前++还是后++,最终都是使得变量自身的值+1
  • 例如: int num = 10;
  • 前++: ++num; =====>导致num的值变成11
  • 后++: num++; =====>导致num的值变成11
  • 参与操作使用:
  • 如果是前++,那么是先让变量自身的值+1,再取变量的值去参与操作
  • 例如: int num = 10;
  • 前++: int res = ++num; =====> res: 11 , num: 11
  • 如果是后++,那么是先取变量的值去参与操作,再让变量自身的值+1
  • 例如: int num = 10;
  • 后++: int res = num++; =====> res:10 , num: 11
  • 自减: –
  • 运算规则如上(把+换成-)

逻辑运算符

  • & 逻辑与 遇false则false,只有2边同时为true,结果才为true

  • | 逻辑或 遇true则true,只有2边同时为false,结果才为false

  • ^ 逻辑异或 相同为false,不同为true

  • ! 逻辑非(取反) true变false,false变true

  • 作用: 逻辑运算符可以用来连接多个关系表达式

  • 结论: 逻辑表达式的结果一定是boolean类型

  • ```
    public class Deom1{
    public static void main(String[] args) {
    //定义变量
    inti=10;
    intj=20;
    intk=30;

    //& 有false则false
    System.out.println((i>j) &(i>k)); //false & false
    System.out.println((i<j) &(i>k)); //true & false
    System.out.println((i>j) &(i<k)); //false & true
    System.out.println((i<j) &(i<k)); //true & true
    System.out.println(“——–”);

    //| 有true则true
    System.out.println((i>j) |(i>k)); //false | false
    System.out.println((i<j) |(i>k)); //true | false
    System.out.println((i>j) |(i<k)); //false | true
    System.out.println((i<j) |(i<k)); //true | true
    System.out.println(“——–”);

    //^ 相同为false，不同为true
    System.out.println((i>j) ^(i>k)); //false ^ false
    System.out.println((i<j) ^(i>k)); //true ^ false
    System.out.println((i>j) ^(i<k)); //false ^ true
    System.out.println((i<j) ^(i<k)); //true ^ true
    System.out.println(“——–”);

    //!
    System.out.println((i>j)); //false
    System.out.println(!(i>j)); //!false
    System.out.println(!!(i>j)); //!!false
    System.out.println(!!!(i>j)); //!!!false
    }
    }

三元运算符

  • 格式: 结果为boolean类型的表达式 ? 值1 : 值2;
  • 执行流程:
  • 首先计算表达式的结果
  • 如果表达式的结果为true,那么值1就作为这个三元表达式的结果
  • 如果表达式的结果为false,那么值2就作为这个三元表达式的结果
  • 案例:获取两个数据中的较大值

数据输入

  • 导包: import java.util.Scanner;
  • 创建对象: Scanner sc = new Scanner(System.in);
  • 接收整数数据: int x = sc.nextInt();
  • 扩展:接收小数数据: double d = sc.nextDouble();
  • 扩展:接收字符串数据: String str = sc.next();
  • 注意:
  • 以上步骤都是固定格式,能改变的就是sc和x变量名,其余都不可以改变
  • 以上代码只能接收整数
  • 导包是在类的上面进行导包,也就是最上面一行

顺序结构

  • 按照代码的顺序从上往下依次执行

if结构

  • 格式:
    if(关系表达式){
    语句体;
    }
  • 执行流程:
  • 首先计算关系表达式的结果
  • 如果关系表达式的结果为true,那么就执行语句体
  • 如果关系表达式的结果为false,那么就不执行语句体
  • 执行完整个if结构,程序会继续往下执行…

if…else…结构

  • 格式:
    if(关系表达式){
    语句体1;
    }else{
    语句体2;
    }
  • 执行流程:
  • 首先计算关系表达式的结果
  • 如果关系表达式的结果为true,那么就执行语句体1;
  • 如果关系表达式的结果为false,那么就执行语句体2;
  • 执行完后,程序继续执行后面的代码…

if…else if…else结构

  • 格式:
    if(关系表达式1){
    语句体1;
    }elseif(关系表达式2){
    语句体2;
    }
    …
    elseif(关系表达式n){
    语句体n;
    }else{
    语句体n+1;
    }
  • 执行流程:
  • 首先计算关系表达式1的结果
  • 如果结果为true就执行语句体1,如果为false就计算关系表达式2
  • 如果结果为true就执行语句体2,,如果为false就计算关系表达式3
  • …
  • 如果所有的关系表达式的结果为false,那么就执行语句体n+1
